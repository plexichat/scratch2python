# scratch2python - dependencies
#
# This project has NO third-party Python dependencies.
# It uses only Python 3 standard library modules:
#   argparse, ast, collections, importlib, json, logging, math, os,
#   pathlib, py_compile, re, shutil, stat, sys
#
# The generated output code (main.py) also uses only stdlib:
#   tkinter, time, traceback (and conditionally ctypes, atexit on Windows)
#
# Requirements: Python 3.10+
